# 4chan image archiver
handy web browser bookmarklet to archive all 4chan images.

# how to use
copy the whole content of [4chan-image-archiver.js](https://raw.githubusercontent.com/eightynine77/4chan-image-archiver/refs/heads/main/4chan-image-archiver.js) file and paste it as a new browser bookmarklet. you simply have to create a new bookmarklet and then paste the whole content of 4chan-image.archiver.js into that bookmarklet. then you can use that bookmarklet
